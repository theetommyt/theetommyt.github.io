Icons

Basic icons

Add Apollo icons to a page using the .icon base class, followed by the character-specific .icon-{name} modifier class.

Icon color

Change the color of an icon using the color property, as you would for changing the color of text.

Icon size

To change the size of an icon, simply change the font-size property of the icon itself or its parent element.

Vertical alignment

Icons are aligned with inline text using the vertical-align property. In most cases, the default alignment works well, but here are some examples of how to modify the alignment using different vertical-align values or relative positioning.

Forms

Basic form

Use forms to collect information from users, such as contact information, status updates, or credentials.

HTML5 input types

Be sure to use the appropriate type attribute that corresponds to the type of information being collected (reference).

Form with grid

To make form labels appear alongside their corresponding controls, you can use grid columns and text alignment classes to create the desired layout. The column width for the labels and controls should be made to accommodate the length of your labels at each breakpoint. Labels should stack with controls for mobile screen sizes.

Available input controls

Several input controls are available to choose from. Each one has a specific intended use and offers unique interaction patterns that are familiar to users.

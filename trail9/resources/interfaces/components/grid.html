Responsive columns

The grid is responsive using the mobile-first approach. So you start by defining the column width for the xs screen size, then again for each screen size where you want the column width to change. If you skip a screen size, it will automatically inherit the size from the next smaller size that was specified. For example, .col-xs-12 .col-md-6 would apply 12 column width to sm screens since the next smaller size is .col-xs-12.

Column Offsets

Grid columns can be offset as needed to shift content or visually center a column. As with everything else in the grid, these offsets can be applied on a per screen size basis.

Push and pull

Apollo UI

HTML and CSS for the Apollo visual language.

Version: 1.0.0-beta.3

GitLab Project

Project in development

This project is currently in development. The markup, styles and documentation are subject to change without notice until the first official release. In the meantime, feel free to experiment with these components and report any issues you discover using the Issue Tracker.

Package management

Use Apollo UI as a dependency of your application with NPM or Bower. This gives you access to the variables, mixins and other goodies so you can make your application-specific styles consistent with the rest of the framework.

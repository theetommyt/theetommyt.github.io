Fluid Jumbotron

To eliminate the default side padding in favor of fluid container padding, add the .jumbotron-fluid modifier class with a .container-fluid inside.

No side padding

Ok, well there is side padding, but not on the Jumbotron itself. The side padding here comes from the .container-fluid inside the jumbotron.

Background images

When using an image background, it's a good idea to add a semi-opaque overlay to improve readability. Below are examples of how to use CSS to add a light or dark overlay to a background image.
Light overlay

Light background image

This example uses a background image with a semi-transparent white linear gradient to add a light color mask to the background image and ensure readability.

Dark overlay

Dark background image

Adding the bg-dark class takes care of inverting the text, and then the same linear gradient trick is used to add a dark color overlay to ensure that the white text stands out.

Alerts

Alert variants

Alerts are dismissable messages that give users feedback when in-page events occur. Use them to indicate whether a process was successful or if an action may have an adverse outcome.

Information
New options available.
Success
Changes saved.
Warning
Items need attention.
Danger
Unable to save changes.
Bottom positioning

Adding the .alert-bottom class to any alert message reverses the position of the fade-out shadow. Use this when your alert message is shown at the bottom of a page or section.
